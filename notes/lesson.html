The JSON methods

Unit 2: Week 17 - Lesson 2

Since javascript and json go hand-in-hand, the javascript developers created a method specifically for working with objects stored in .json files.

the JSON object - a special javascript object with methods designed to transform and utilize objects from a .json file.

Since .json files must be small and easy to transmit across the internet, javascript must be able to translate objects from a .json file from their string form and into an actual javascript object. To do this, we use JSON.parse().

JSON.parse() - a method used to translate text from a .json file and into a proper javascript object.

// example 1 - using text recieved from a .json file

var boat = JSON.parse('{"name": "titantic", "carryCapacity": "2435", "inOperation": "false"}')

console.log(boat.name)
// output - 
// titantic

JSON.stringify() - a method used to convert a javascript object into a string for .json storage. 

// example 2 - converting a javascript object into text for a .json file

var boat = {name: "titantic", 
carryCapacity: 2435, 
inOperation: false
}

var jsonReadyText = JSON.stringify(boat)

console.log(jsonReadyText)
// output - 
// '{"name":"titantic","carryCapacity":"2435","inOperation":"false"}'

We can now efficiently communicate data across applications and servers!
